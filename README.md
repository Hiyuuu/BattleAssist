# BattleAssist
バトルロワイヤルのspigotプラグイン

## 仕様

対応サーバーソフトウェア : PaperMC
対応バージョン          : 1.16.5

## ダウンロード

[リリース](https://github.com/riyaya1528/BattleAssist/releases/tag/1.0)からダウンロードができます

## コマンド

`/start`   ゲームを開始します

`/stop` ゲームを終了します
